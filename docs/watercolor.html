Watercolor

Quickstart

Watercolor adds labels for each a11y element on the screen for mouse-free interaction.

  • git clone https://github.com/c-Loftus/watercolor ~/.talon/user
  • sudo apt-get install libgirepository1.0-dev gir1.2-atspi-2.0 #(switch by distro)
  • cd ~/.talon/user/Watercolor
  • make run

Commands

You must have the Watercolor server running to use these commands.
All letter names use the Talon phonetic alphabet

color toggle Turns watercolor on and off.
color {two letter names} color air bat clicks the label "AB"
inspect {two letter names} inspect air bat print info on the label "AB"
