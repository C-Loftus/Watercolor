Watercolor

Quickstart

Watercolor adds labels for each a11y element on the screen for mouse-free interaction.

Installation
  • git clone https://github.com/c-Loftus/watercolor ~/.talon/user
  • cd ~/.talon/user/Watercolor
  • make run

Commands

You must have the Watercolor server running to use these commands. All letter names use the Talon phonetic alphabet

" color toggle " turns watercolor on and off.
" color {two_letter_names}" Example: " color air bat " clicks the label "AB"
" inspect {two_letter_names}" Example: " color air bat " Print info on the element with label "AB"
